# ProbePad
**Privacy-first API testing tool for iPhone & iPad**

ProbePad is a clean, fast, and local-only HTTP API tester designed for developers who want to inspect requests and responses on the go.

✅ No accounts  
✅ No tracking  
✅ No analytics  
✅ All data stays on your device  

---

## What you can do with ProbePad
- Create and save HTTP requests
- Configure method, URL, headers, and body
- Execute requests and inspect responses (status, headers, body, timing)
- Browse response history for each request
- Import/export requests as **cURL** and **JSON**
- Clear all data anytime

ProbePad is built to be simple, reliable, and respectful of your privacy.

---

## Responsible use
ProbePad is intended for testing your own APIs and debugging endpoints.  
The app includes rate limiting to help prevent accidental abuse.
