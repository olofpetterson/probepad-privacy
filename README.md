# ProbePad Privacy Policy

This repository hosts the public **Privacy Policy** for the iOS app **ProbePad** via GitHub Pages.

## Live Privacy Policy
The privacy policy is published here:
**https://olofpetterson.github.io/probepad-privacy/**

## About ProbePad
ProbePad is a privacy-first HTTP API testing tool for iPhone and iPad.
The app does not require sign-in and does not include analytics or tracking.

## What’s in this repo
- `index.md` (or `index.html`) – the public privacy policy page

## Publishing
This repo is deployed using **GitHub Pages**:
- Source: `main`
- Folder: `/ (root)`

To update the policy:
1. Edit `index.md` 
2. Commit to `main`
3. GitHub Pages will automatically redeploy within a minute
