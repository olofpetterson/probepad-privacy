## About ProbePad
ProbePad is a privacy-first HTTP API testing tool for iPhone and iPad.

- No account required
- No analytics
- No tracking
- All data stays on-device

---

## Privacy Summary
ProbePad does not collect, store, or share any personal data with the developer or any third parties.

- Requests and responses are stored locally on your device
- The app only performs network requests that you explicitly execute
- You can delete all stored data at any time from Settings

---

## Support / FAQ

### Does ProbePad collect any data?
No. ProbePad does not collect or transmit any user data to the developer.

### Does ProbePad use analytics or tracking?
No. ProbePad does not include analytics SDKs or tracking.

### Where is my data stored?
All requests and responses are stored locally on your device.

### Can I delete my data?
Yes. Go to **Settings** in the app and use:
- Clear Requests
- Clear Response History
- Clear Everything

### Does ProbePad support HTTP (insecure)?
ProbePad supports **HTTPS only**.

### Why does a request fail?
Common reasons:
- Invalid URL format
- No internet connection
- Timeout / server not responding
- SSL/TLS issues
- The target API blocks the request

### How do I report a bug?
Please email: **your@email.com**  
Include:
- iOS version
- device model
- steps to reproduce
- screenshots (if possible)
